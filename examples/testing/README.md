# Test Example

This example shos how to test application code with goka.
It can be executed with `go run main.go` but it does not do much, because no
events are produced by default. It is more about the test code.

# How to test with goka

Testing application code is supported in two ways:

* write unit tests mocking goka.Context
* write integration tests mocking the communication with Kafka.
